How to Use Spiddyyy Carousel

Follow these steps to add the Spiddyyy Carousel to your website. It's lightweight, responsive, and supports custom HTML slides and image-based navigation.

1. Include the CSS

Add the minified CSS from the Spiddyyy CDN in your ``.

<link rel="stylesheet" href="https://cdn.jsdelivr.net/gh/Spiddyyy/Slider-carousel-cdn@latest/spiddyyy-carousel.min.css">

Configuration Options

Customize the carousel behavior using the `carouselConfig` object.

Property Description Default
width Carousel width (e.g., '100%', '600px', '80vw') Viewport width
height Carousel height (e.g., '400px', '50vh') Viewport height - 150px
autoplayInterval Time between slides in ms 2000
transitionSpeed Slide transition speed (e.g., '0.5s') 0.5s
buttonBg Navigation button background rgba(0, 0, 0, 0.5)
buttonColor Navigation button color white
showNavButtons Show/hide navigation buttons true
prevButtonContent HTML content for previous button <img src="..." alt="Previous" loading="lazy">
nextButtonContent HTML content for next button <img src="..." alt="Next" loading="lazy">
autoplay Enable/disable autoplay true

Performance Tips

  • Use modern image formats (e.g., WebP) and srcset for responsive images.
  • Ensure your images are optimized and include loading="lazy".
  • Preload the carousel CSS if it's above the fold: <link rel="preload" href="..." as="style">.
  • Sanitize dynamic HTML in slides or button content to prevent XSS (e.g., using DOMPurify).
