Configuration Options

Customize the Spiddyyy Carousel with these `carouselConfig` properties.

Property Description Default
width Carousel width (e.g., '100%', '600px', '80vw') Viewport width
height Carousel height (e.g., '400px', '50vh') Viewport height - 150px
autoplayInterval Time between slides in ms 2000
transitionSpeed Slide transition speed (e.g., '0.5s') 0.5s
buttonBg Navigation button background rgba(0, 0, 0, 0.5)
buttonColor Navigation button color white
showNavButtons Show/hide navigation buttons true
prevButtonContent HTML content for previous button (e.g., image)
nextButtonContent HTML content for next button (e.g., image)
autoplay Enable/disable autoplay true

Performance Tips

Optimize your website when using Spiddyyy Carousel:

  • Use WebP images and srcset for responsive image loading.
  • Ensure images include loading="lazy" to defer offscreen loading.
  • Preload the carousel CSS for above-the-fold carousels: <link rel="preload" href="..." as="style">.
  • Sanitize dynamic HTML in slides or button content to prevent XSS (e.g., using DOMPurify).
  • Minify your carouselConfig script in production.
